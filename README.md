Este projeto é uma aplicação simples em PHP que consome a API pública ViaCEP para buscar endereços a partir de um CEP informado pelo usuário. O sistema filtra e valida o CEP digitado, realiza a requisição à API e preenche automaticamente os campos de logradouro, bairro, cidade e estado no formulário. Também trata casos de CEP inválido ou não encontrado, exibindo mensagens adequadas ao usuário.
